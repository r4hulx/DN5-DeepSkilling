# SRP (Single Responsibility Principle)

## Objective
Demonstrate the Single Responsibility Principle using a simple Employee salary calculator.

## Problem
A single class performing multiple responsibilities becomes difficult to maintain.

## Solution
Responsibilities are divided into separate classes:
- Employee → Stores employee information
- SalaryCalculator → Calculates annual salary

## Output

Employee : Rahul
Monthly Salary : 50000
Annual Salary : 600000
